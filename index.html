JavaScript的比较运算符

==、!=、===、！== 的区别

联想到的知识：函数声明和函数表达式的主要区别：

函数声明会进行函数变量的提升

fun1();//“fun1”
function fun1(){
console.log(“fun1”);
}

函数声明则不会出现变量提升

fun2();
var fun2 = function(){
console.log(“fun2”);
};

此时会报错

只有定义变量fun2后调用才不会报错
var fun2 = function(){
console.log(“fun2”);
};
fun2();

总结

1.==和 != 比较若类型不同，先偿试转换类型，再作值比较，最后返回值比较结果 。

2.=== 和 !== 只有在相同类型的情况下，才会进行值比较。
